## 一、项目介绍

基于 Halo 2.x 提供的 API 接口，为微信小程序提供的一套开源的博客应用。

- 完全免费开源，包括程序源码、插件源码
- 页面支持插件配置
- 使用最新流行的技术栈
- 支持特色功能，恋爱日记
- 理论支持编译为 小程序（推荐）、APP

### 功能介绍

- 基础功能：文章列表、文章分类、文章详情、个人图库、友情链接、我的瞬间

- 特色功能：恋爱日记，精心为您设计了秀恩爱的模块，秀出关于你们的故事、情侣甜蜜相册、恋爱记录清单，为您的恋爱保驾护航。
